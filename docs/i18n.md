# Copy and localization

Calorie Flow currently ships in English only; there is no translation layer. Keep user-facing copy centralized near the relevant feature and avoid introducing a partial localization abstraction.

If another locale is added, introduce a single translation system and migrate all user-visible strings for the affected surfaces in the same change. Nutrition units, dates, and decimal formatting must follow the selected locale.
